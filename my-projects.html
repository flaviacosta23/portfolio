Projeto de site de compras de equipamento de camping, feito com Html, Css e JavaScript. No site tem a função de mostrar quantos itens estão na cesta, o valor somado da compra, o valor do envio e a soma total da compra adicionada ao envio.

Projeto de página de login, desenvolvido para o desafio do Frontend Mentor, feito com Html, Css e JavaScript. Ao fazer o login com as informações corretas o site indica que o loging foi feito, caso contrário as bordas das caixas de inserir os dados ficam vermelhas e aparece os ícones de erro.
